# 🏥 AI Medical Device Chatbot

An intelligent **AI-powered Medical Device Information System** built using **Flask**, **Machine Learning**, and **Supabase**. The application enables healthcare professionals, students, and researchers to explore medical devices, ask natural language questions, and access structured device information through a modern web interface.

---

## 📌 Overview

The AI Medical Device Chatbot combines **Machine Learning(ML)** with a cloud-based database to provide quick and reliable information about various medical devices.

Users can:

* 🤖 Chat with an AI assistant about medical devices
* 🔍 Search and browse a medical device library
* 📄 View detailed device information
* 🕒 Access chat history
* 🔐 Register and log in securely
* ☁️ Store user data using Supabase
